Šta je zemlja bez svoje infrastrukture? Anonutopia ima mnogo besplatnih projekata i aplikacija otvorenog koda koje imaju za svrhu upravo postavljanje svoje infrastrukture a sve u cilju dobijanja profita koji će služiti za finansiranje univerzalnog osnovnog dohodka.

Ovo su neki od njih:


ANOTA

Naš glavni proizvod je - ANOTA. Ona je decentralizovani stabilni novčić programiran uz pomoć teorije igara (Game theory) , pametnih ugovora (smart contracts) i LPoS(Leased Proof of Stake) Oracles - Anon0racle. On je takođe investicioni bankarski token. Pročitajte više o ANOTI ovde.


Cripto Ekvivalent Fiat Valutama

Anonutopjini centralzovani stabilni novčići - CryptoEuro, CryptoDollar, itd.


AnonTaxi

Anonutopijini projekat deljenja vožnje, iznajmljivanja vozila i taksi aplikacija.

GitHub


AnonATM

Cheap, accessible and Open Source crypto ATM that uses our Open Source software. Still under the development.


AnonNovčanik

Anonutopijin novčanik zasnovan je Waves novčaniku. Za sada radi samo kroz internet pregledač, ali pripremamo Android i IOS aplikacije.

GitHub


AnonMenjačnica

Naša kriptoi fiat menjačnica prikačena na Waves DEX u pozadini. I dalje je u razvoju.


AnonOracle

Osnova našeg bankarskog sistema, koriste LPoS(Leased Proof of Stake) za dostavljanje korisnih spoljnih podatka, i drže naš fond u novčanicima sa višestukim potpisima.

GitHub


GoWaves

Waves platofrma u Golang bibliotekama.

GitHub


WavesCLI

Waves CLI platforma (Command Line Interface).

GitHub


AnonPortal

CMS (Content Management System) za našu mrežu web sajtova, može se koristiti za manji novinski portal.

GitHub


AnonNet

Anonutopijina Ethereum blockchain mreža. Privatna za sada, nakon testiranja biće javna.
